# spring-web-with-h2

Projeto apenas para estudos.

Ferramentas utilizadas nesse projeto:

 - Spring Web - facilita o processo de desenvolvimento com conceitos de Inversão de Controle e Injeção de Dependências.
 
 - Spring DevTools - reinicia o aplicativo quando faço mudanças no código, para agilizar o processo de desenvolvimento.

 - JPA - uma API do Java que descreve uma interface comum para frameworks de persistência de dados. A JPA define um meio de mapeamento objeto-relacional para objetos Java simples e comuns, denominados beans de entidade.
 
 - Hibernate - framework para o mapeamento objeto-relacional (MOR), trabalha em cima da JPA.

 - H2 database - banco de dados relacional escrito em Java, possui suporte a persistência de dados na memória.

 - Thymeleaf - para servir XHTML / HTML5 na camada de visualização de aplicativos Web baseados em MVC. Traduz os objetos Java para o HTML.

 - Maven - ferramenta de automação e gerenciamento de projetos.

 - Maven Wrapper - permite utilizar somente uma versão especifica do Maven na minha equipe de desenvolvimento independente das versões instaladas nas outras maquinas, com isso podemos executar o projeto Maven sem ter o Maven instalado (Lembrar ./mvnw)
